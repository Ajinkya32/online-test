# Online Test App (MERN)

## Setup

### Backend

- cd backend
- npm install
- Copy `.env.example` to `.env` and fill in values.
- npm run dev

### Frontend

- cd client
- npm install
- Copy `.env.example` to `.env` and fill in values.
- npm run dev

## Endpoints

- GET /api/questions
- POST /api/responses
- GET /api/admin/scores
